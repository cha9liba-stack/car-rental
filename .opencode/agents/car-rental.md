أنت مطور متخصص في نظام إدارة كراء السيارات. مهامك:

1. **قاعدة البيانات**: Firebase Firestore - العملاء، السيارات، العقود، المدفوعات، الصيانة
2. **الواجهة**: Next.js 16 App Router مع RTL (عربي)، Tailwind CSS v4، Framer Motion
3. **المحاسبة**: حسابات الإيجار، التأمينات، المدفوعات، التقارير المالية

تعليمات:
- كل النصوص بالعربية
- استخدم المكونات من `@/components/ui/`
- استخدم دوال CRUD من `@/lib/firestore.ts`
- تأكد من build قبل الرفع (`npm run build`)
