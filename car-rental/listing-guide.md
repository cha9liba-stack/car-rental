### Description

AutoRent is a fully-featured car rental management system built with Next.js 16, Firebase, and Tailwind CSS. It's designed for car rental agencies in Tunisia and North Africa, with built-in support for Tunisian CIN/document formats, Dinar currency, and Arabic/French bilingual interface.

The system includes everything a rental agency needs: client management, fleet tracking, contract generation, payment processing, maintenance scheduling, expense tracking, loyalty program, and comprehensive reporting.

**Key Modules:**
- Dashboard with real-time stats and alerts
- Client management (CIN, driver license, contact info)
- Fleet management with images, pricing tiers, insurance/inspection tracking
- Contract management with detailed driver info, financial breakdown, digital signatures
- Payment tracking (cash/card/transfer, deposits, returns)
- Maintenance scheduling with garage tracking
- Expense tracking by category
- Fleet overview with real-time status
- Loyalty program (points, levels: Bronze→Silver→Gold→Platinum)
- Activity log / audit trail
- Reports with charts (monthly revenue, fleet status, top clients)
- Dark mode + RTL support
- Multi-language (Arabic + French)

### Tech Stack

- **Frontend:** Next.js 16 (App Router), TypeScript, Tailwind CSS v4, Framer Motion, Recharts
- **Backend:** Firebase Firestore, Firebase Authentication, Firebase Storage
- **UI:** Custom components (glassmorphism design), Lucide icons, Sonner notifications
- **Forms:** React Hook Form + Zod validation
